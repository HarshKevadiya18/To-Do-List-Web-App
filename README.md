# Todo List - Task 2

A simple Todo List web application built using **HTML, CSS, and JavaScript**.

---

## Features

* Add new tasks
* Mark tasks as completed using a checkbox
* Delete tasks
* Dynamic task status heading:

  * Shows **"No task to do"** when the list is empty
  * Shows **"Task to do"** when tasks are present
* Input validation to prevent empty tasks
* Responsive and clean user interface
* Hover effects on buttons and input fields

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## How It Works

### Add a Task

1. Enter a task in the input field.
2. Click **Add Task**.
3. The task is added to the list.

### Complete a Task

* Click the checkbox next to a task.
* The task text will be displayed with a strikethrough effect.

### Delete a Task

* Click the **Delete** button next to a task.
* The task is removed from the list.
